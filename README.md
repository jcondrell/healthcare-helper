# healthcare-helper
Website to help facilitate individuals' healthcare experience
By: Jade Condrell & Ryan Taylor
Opened: October 1, 2025


Overview: Healthcare Helper is a website designed to aid patients and healthcare workers to observe general trends in medical data and use the information to guide their own medical experiences.
